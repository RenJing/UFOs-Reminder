=========功能介绍==========

1，监控UFOs系统的最新状态
2，新订餐提醒
3，订餐即将结束提醒
4，页面实时列出所有Open的订餐计划
5，订餐计划的背景颜色表示当前状态
	红色：10分钟内结束
	黄色：30分钟内结束
	绿色：30分钟后结束


=========使用技术==========
Html5：
	Audio：使用了html5提供的新标签<audio>，并使用javascript自动播放音频 
	本地存储：使用html5新功能的localStorage，本地存储当前订餐计划，解决了重启浏览器会重复提醒的问题
	Canvas: 使用了html5提供的新标签<canvas>，既实现了背景的纹理效果，又减少的对图片文件的依赖
css3：
	使用了一些css3的新特性，如圆角，颜色渐变，放大效果
Other:
	JSONP: 解决了跨域Ajax读取数据的问题
